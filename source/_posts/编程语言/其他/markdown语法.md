有了平台之后，还要学习Markdown语法，我把它理解为写文章的笔。
Markdown其实就是一种易于编写的普通文本，会对部分标签的文本进行渲染。在渲染时会转换为html标签。在使用过程中，在语法后至少保留一个空格或空行。参考链接[Markdown 教程](https://www.runoob.com/markdown/md-tutorial.html)
# 标题
Markdown支持6种级别的标题，使用#的数量来表示等级
``` 标题等级
    # 一级标题
    ## 二级标题
    ### 三级标题
    #### 四级标题
    ##### 五级标题
    ###### 六级标题
```
效果如下【为防止目录混乱，此处使用图片展示】：
![标题等级](markdown语法/标题等级.png)
# 列表
- 无序列表，使用*、+、-表示
- 有序列表，使用数字加.标识，例如：1.
    为了方便编号，有序列表只关注第一个项目的数字编号，例如：
``` 列表显示
    * 选项A
    * 选项B
    * 选项C
    1. 选项1
    1. 选项2
    2. 选项3
```
效果如下：
> * 选项A
> * 选项B
> * 选项C
> 1. 选项1
> 1. 选项2
> 2. 选项3
# 强调
在段首增加">"，可以使该段文字高亮显示，本文章中的示例已使用强调显示
``` 文本强调
    > 强调文本
```
效果如下：
> 强调文本

使用一个\*或\_包裹文字，可实现斜体；使用两个\*\*或者\_\_包裹文字，可以实现粗体
``` 字体强调
    强调：*斜体*、**粗体**、**粗斜体***
```
效果如下：
> *斜体*、**粗体**、***粗斜体***
# 分割线
在空行使用三个以上的***或者---即可
``` 分割线示例
    *****
    -----
```
效果如下：
> *****
> -----

# 表格
Markdown 制作表格使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行。
- ---: 设置内容和标题栏居右对齐。
- :--- 设置内容和标题栏居左对齐。
- :---: 设置内容和标题栏居中对齐。
``` 表格示例
    | 左对齐 | 右对齐 | 居中对齐 |
    | :-----| ----: | :----: |
    | 单元格 | 单元格 | 单元格 |
    | 单元格 | 单元格 | 单元格 |
```
| 左对齐 | 右对齐 | 居中对齐 |
| :-----| ----: | :----: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |
# 代码块
使用反引号(\`)，键盘左上角)包裹一段代码，也可以使随便一段文字。 使用\`\`\`包裹一段代码，并指定一种语言时，可以根据语言风格渲染颜色。
```
    \```C
    printf("hello world!")
    \```
```
效果如下：
```C
printf("hello world!")
```
