鸟机【1055】

【4.8】第一轮

①油门值、电压-转速

ns = U*(-0.0373*dp² + 11.8*dp - 93.222)

dp：油门值（百分值） U：电压（伏特） ns：转速（rpm）

②前进比：桨前进速度与桨尖速度比值

J=V/ns*D

V：空速 ns：转速（RPM*2PI/60）rad/s D：螺旋桨直径（米）

③J-CT

CT= -0.0623*J² - 0.004*J + 0.0028

T = CT*rho*ns²*D^4

rho：kg/m³ ns：rad/s D:m

④J-CP

CP = -0.0471*J² + 0.006*J - 2E-05 【y = -0.3339x3 + 0.0864x2 - 0.0052x + 0.0002】

P = CP*rho*ns³*D^5

rho：kg/m³ ns：rad/s D:m

⑤J-桨效率

y1 = -66.302*J² + 17.947*J - 0.0759

⑥RPM-电机效率

y2 = -2E-08*RPM² + 0.0003*RPM - 0.4484

⑦总功率：

T*V/y1/y2

修改数据：

③J-CT

y = -0.0041x² - 0.0101x + 0.003

⑥RPM-电机效率（加入0截距）

y = -4E-12x3 + 5E-08x2 - 3E-05x

【4.10】

①油门值、电压-转速

ns = (1.119 * dp + -0.001606 * dp .^2)*(0.2668 + 0.1056 * U)

dp：油门值（百分值） U：电压（伏特） ns：转速（rad/s）

②前进比：桨前进速度与桨尖速度比值

J=V/ns/D

V：空速 ns：转速rad/s D：螺旋桨直径（米）

③J-CT

CTn = -0.08997 * ARn.^2 -0.05982 *ARn +0.1262;

T = CT*rho*ns²*D^4

④J-CP

CPn = -0.04899 * ARn.^2 -0.0005172 * ARn +0.05093;

P = CP*rho*ns³*D^5

⑤RPM-电机效率

eta_emotorn = -5.575e-05 * nn.^2 +0.01122 * nn + 0.2199;

⑥总功率：

T*V/y1/y2

【Final】

转速RPM，ns=RPM/60=rad/s

①油门值、电压-转速

ns = (-0.001398*dp² + 0.9793*dp) * (0.1198*U + 0.3139)

dp：油门值（百分值0-100） U：电压（伏特） ns：转速 （rad/s）

②前进比J：桨前进速度与桨尖速度比值

J=V./ns/D

V：空速m/s ns：转速rad/s D：螺旋桨直径m

③J-CT

CT = -0.08997*J² - 0.05982*J + 0.1262;

J>0.8 : CT = 33.627732440735322 * exp(-9.237330525654857*J);

T = CT*rho*ns²*D^4

【CT(J>0.8) = 33.627732440735322 * exp(-9.237330525654857 * J(J>0.8))】

rho:空气密度（kg/m³）

④J-CP

CP = -0.04899*J² -0.0005172*J +0.05093

J>0.9 ： CP = 17.555784504963174 * exp(-8.215449239738124*J);

P = CP*rho*ns³*D^5

【CP(J>0.9) = 17.555784504963174 * exp(-8.215449239738124 * J(J>0.9));】

⑤ns-电机效率

e_motor = -5.575e-05*ns² + 0.01122*ns + 0.2199

⑥总功率：

P_all = P / e_motor +5
